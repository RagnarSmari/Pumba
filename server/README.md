# Pumba server side
Stack:  
* [Gin](https://gin-gonic.com/) - for web framework
* [Zap](https://github.com/uber-go/zap) - for logging

## Env file structure

```

DATABASE_PORT=""
DATABASE_USER=""
DATABASE_USER_PASSW=""
DATABASE_NAME=""
GOOGLE_OAUTH_CLIENT_ID=""
GOOGLE_OAUTH_CLIENT_SECRET=""
```
